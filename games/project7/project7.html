Metal Detecting prototype

Metal Detecting simulator with photoscanned environment assets.

About the Project

Inspired by a hobby of mine, I started this metal detecting themed project. I learned photogrammetry and turned some trees, rocks and similar natural objects into 3D assets to use primarily in this project. I studied Agisoft Metashape and found it to be a really powerful tool for creating realistic 3D assets from photos. I made a table of possible finds that helped me randomly generate them in the game world with controllable varity. Some basic UI was also experimented with, along with a crude inventory system. I noly used Unreal Engine's blueprints in this project. They can look quite messy as can be seen in the screenshots below, but blueprints are a really fast and easy way to prototype features. For intensive calculations or heavy for-loops I prefer C++ though.
