StrawBarrier (Game Jam)

My submission for the GMTK Game Jam 2024. Theme of the jam was "Built to scale" Link to the game's itch.io page: https://ieevari.itch.io/strawbarrier

About the Project

In this game you need to protect a strawberry comet until help arrives. The game is not very polished and has a few bugs still, but I'm happy how it turned out. The game jam lasted for 4 days, but I could only work on the game for 2 days. I'm most proud of the design idea of working on different "levels" at the same time. I was not super happy about the visual style of the game. Sound design was fun and I really like how the sound changes with the different levels of zoom. Overall, this project is not my personal favourite but I like game jams so the process was enjoyable!
