Intuition (test project)

Abandoned project that was mostly used to learn Unreal Engine, low poly art and animation.

About the Project

This project was a learning experience aimed towards learning the basics of Unreal Engine 4. I focused on learning the UE's powerful blendspace animation workflow and learned more about low poly modeling and importing meshes to UE. I also learned the easy to use blueprint visual coding feature that UE has. The 3D models were made using Blender.
