Jatsi (a Yahtzee variant)

A modified ruleset for the popluar dice game Yahtzee. Supported platforms are PC, Mac and Android.

Screenshots

About the Game

This game was developed as a hobby project and to make a custom ruleset, not found anywhere online, to be playable on Desktop and Mobile. The game is playable and should be bug-free, but as it lacks a proper tutorial and the rules are somewhat hard to grasp it has not been made public yet.
