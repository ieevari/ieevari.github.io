Ant simulator

A game where I experimented with dynamic animations.

About the Project

I wanted to learn more about IK animations in games, as I had already used IK rigs in 3D modeling. Unreal Engine has a lot of helpful tools already built-in, and one of those is the Control Rig. Setting up the Control Rig for the ant character was relatively easy and the basic IK setup is straight forward. However, implementing things like the foot rising up when moving forward or the prediciton of the next step are increasingly harder to pull off in a realistic way.
