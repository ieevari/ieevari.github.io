Here are some of my interactive 3D models.

Garrett ACE 300i

A detailed model of my first metal detector. Created in Blender, textured in Substance Painter.

Cod Fish

A Cod model prototype I made for a fishing game I worked on 2022. Textures were made with the help of Stable Diffusion, using an image from the internet as a source.

Robot Mouse

A mechanical mouse I modeled to learn real-time IK animations in Unreal Engine. Textures were generated in-engine, so they can't be seen here.

Robot Rover

Rover with triangular tracks I made for a prototyping track based movement in games.

Yellowfoot (suppilovahvero)

Yellowfoot mushroom I modeled and textured using photos as a reference and by projecting textures onto the model. This was done before I learned to do photoscans but the result is pretty good!

Bee

Bee made to act as an enemy in a group game project I was a part of.

Man

A character model with animations. Used in 2D graphics by rendering frame-by-frame into pixel art style.

Tree Stump

My own photo scan of a tree stump. Made for a game project. Game ready asset. Programs used: Blender and Agisoft Metashape.

Plant

My own photo scan of a plant that looks like wild strawberry. Made for a game project. Game ready asset. Programs used: Blender and Agisoft Metashape.

Tree

My own photo scan of a tree. Made for a game project. Game ready asset. Programs used: Blender and Agisoft Metashape.

Rock

My own photo scan of a rock and surrounding ground. Made for a game project. Game ready asset. Programs used: Blender and Agisoft Metashape.

Soft plastic bait mold

A custom 3D printable mold for inject molding soft plastic baits for fishing. This design worked well, but the plastic deteriorates fairly quickly. It's better to print the negative of the mold and make the mold from something more heat resistant!

Soft plastic shrimp mold

A custom 3D printable mold for inject molding soft plastic baits for fishing. This bait was designed to catch Cod in Norway! (So far it has not succeeded...)

Russian officer's button

A neat metal detecting find photoscanned and digitalized. The button portrays a two-headed eagle. The factory that made these buttons shut down in late 19th century.

Piece of a metal container

An interesting metal detecting find photoscanned and digitalized. It is most likely a broken piece from container of some sort.
